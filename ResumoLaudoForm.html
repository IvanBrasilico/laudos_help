Receber Laudos


Tela para informar recebimento de laudos vinculados ao SAT/LAB e carregar o respectivo arquivo digitalizado (PDF ou imagem) no Banco de Dados.



Nesta tela estão disponíveis quatro ações:

1. "Cadastrar Novo" - Para realizar um novo cadastro.
2. "Salvar edições" - Salva as edições realizadas.
3. "Carregar" - Carrega novos PDFs para a SAT/LAB.
4. "Visualizar" - Permite a visualização do PDF.

CAMPOS DISPONÍVEIS

Nº SAT - Informe aqui o número do SAT/LAB para carregar os arquivos.
Número do Laudo - Exibe o número de laudos que a SAT/LAB possui.
Data - Enforma a data de cadastro do SAT/Lab.
Data da ciência - Exibe a data em que foi tomado conhecimento.
