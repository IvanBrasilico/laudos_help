Tela para Avaliação de Laudos

Esta tela fornece as ferramentas para preparo da Revisão Aduaneira, baseada nos resultados dos Laudos Laboratoriais.



Ações principais:

1. "Arquivar SAT/LAB - Sem Divergência"- O SAT/LAB é arquivado sem divergência. Nesta pasta o pedido e seus atributos não poderão mais serem editados.
2. "Avalidado - Com Divergência"- O SAT/LAB é arquivado numa pasta que sinaliza divergência. A avaliação de Laudo poderá prosseguir/ser editada nesta pasta, somente será bloqueada a edição após a distibuição para autuação.


Aba "Informar divergência e palavras chave":

"Importar palavras chaves" - abre uma janela para a importação de palavras chave diretamente do arquivo PDF do Laudo (abaixo).



"Incluir avaliação de divergências"- Realiza a inclusão de uma linha com as palavras chave escolhidas e a informação se laudo é divergente. 
"Enviar solicitação ao DW"- Roda um relatório no DW, procurando importações anteriores para o mesmo CNPJ Base e palavras chave escolhidas.


Aba "Cadastrar Base de cálculo"



1. Carregar planilha- Abre o diretório para carregar uma planilha. Esta planilha deve ser a gerada pelo DW, com os dados das DIs pesquisadas no DW.
2. Recuperar planilha selecionada - Caso já tenha sido carregada uma planilha, esta ação realiza o download e abertura.

Informar classificação fiscal



1. Incluir Classificação Fiscal - Inclui a classificação fiscal, vinculada ao Usuário que opinou pelo uso desta classificação.
2. Exibir tela de ajuda para classificação fiscal - Exibe o sistema TECRank de auxílio à classificação fiscal;
3. Enviar solicitação ao DW - Roda um relatório no DW, procurando importações anteriores para o mesmo CNPJ Base, palavras chave e NCM escolhidos.

Cadastrar VEL



1. Carregar VEL- Realiza o cadastro do VEL(Valor estimado de Lançamento). Abre o diretório para carregar uma planilha. Esta planilha deve ser a gerada pelo DW, com os dados das DIs pesquisadas no DW. O usuário poderá infomar data de início, fim, e valor total. O sistema guardará estes dados e a planilha produzida em Banco de Dados. O VEL é uma estimativa, baseada nos dados do DWAduaneiro, do valor de lançamento que a revisão aduaneira baseada naquele Laudo poderá gerar.
