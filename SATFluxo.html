Gerenciamento de fluxo de trabalho do SAT/LAB


O sistema trabalha com vários "Estados" possíves para o SAT/LAB, de acordo com o seu andamento no fluxo de trabalho. Esta tela permite a gestão e visualização do SAT/LAB neste fluxo de trabalho.

Criando Pedido - Quando o SAT/LAB ainda está sendo preenchido/editado pelo AFRFB responsável.

Submetido - Terminada a edição, o AFRFB submete o pedido e um PDF assinado ao setor que controla a emissão de laudos. A partir deste momento o SAT não poderá ser mais editado. Se houver necessidade, o setor de Laudo pode selecionar a a ação "Devolver para edição" para retornar ao Estado anterior. Será gerada uma segunda versão do PDF quando o SAT for novamente submetido ao Setor de Laudo.

Enviado para Assistente/LAB - Quando o PDF for enviado ao laboratório ou for designado Assistente, o setor de Laudo deverá informar através da ação "Enviar para laboratório..." e o SAT/LAB ficará nesta pasta para controle. A partir deste momento o SAT/LAB não poderá voltar para edição.

Laudo Recebido - Quando o Assistente/Laboratório entregar o Laudo,  o setor responsável carregará o(s) PDF(s) no sistema  através da ação "Informar recebimento...", e o SAT passará para esta pasta.

Laudo com Divergência - Apresenta todos os laudos que apresentaram divergências em algum dos requisitos.

Impedimento Ativo - quando o laudo possuir um impedimento ativo será adicionado nesta pasta.
Distribuído para autuação - Quando houver necessidade de autuação o laudo será armazenado aqui.

Laudo analisado - Após laudo recebido, Setor/AFRFB poderá realizar uma análise deste, quando então o respectivo SAT/LAB fserá encaminhado para uma destas pastas:

-> Arquivado com Laudo sem resultado - Quando na tela de avaliação de laudo é selecionado Arquivar SAT/LAB - Sem Divergência.

-> Arquivado com Laudo com resultado - Quando na tela de avaliação de laudo é selecionado Avaliado - Com Divergência.

-> Arquivado sem pedido - Quando é realizado um arquivamento prévio do arquivo.

Aguardando manifestação do interessado - Quando o SAT/LAB necessita de uma ação do interessado para continuar seu fluxo.




A tela possui uma visualização em árvore, exibindo os SATs em cada pasta, e, abaixo deles, as seguintes pastas:

Eventos - lista de todas as ações efetuadas no SAT com data e hora
Versões - lista de PDFs criados para o SAT na função "Submeter".
Itens - lista de itens do SAT
Laudos - lista resumo dos laudos recebidos e vinculados a este SAT





CAMPOS DISPONÍVEIS

Arquivamento prévio - Envia o SAT/LAB para a pasta arquivado sem pedido.
Submeter ao GRALT - Submete a solicitação  ao Setor de Laudo.
Devolver para edição - Reenvia o arquivo para a criação de pedido.
Aguardar interessado - O SAT/LAB vai para a pasta "Aguardando manifestação do interessado" até que o interessado execute alguma ação.
Quesitos do importador - Exibe a tela de Quesitos.
Enviar para laboratório - Envia o Assistente/LAB.
Devolver para GRALT - Devolve para o Setor de Laudo.
Anexar/Consultar Laudo - Abre a janela de Anexo e Consulta de Laudo.
Avaliar Laudo - Abre a tela para Avaliação de Laudo.
Arquivar - Arquiva o SAT/LAB.

Ordem de coleta - Abre um PDF com a Ordem de Coleta.
Visualizar SAT/LAB - Abre a janela de Cadastro de visualização do SAT/LAB.
Recuperar PDF SAT/LAB - Abre um PDF para a recuperação do SAT/LAB
Ver Laudo - Exibe o laudo.

Clonar - Faz uma cópia do arquivo.
Aditamento - Clona o arquivo e faz um aditamento ao SAT/LAB.
Impedimento - Realiza o cadastro de impedimento
